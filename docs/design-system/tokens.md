# Tokens

Dark and light themes define semantic background, foreground, surface, border, primary, secondary, status, move-quality, board, radius, and shadow tokens in `apps/web/app/globals.css`. Components consume semantic names rather than direct hexadecimal values.

Core geometry: 4, 6, 8, 12, and 16 px radii; motion principles are 100, 150, 220, 280, and 350 ms.
